## Project Description
This project is an audit of the open source software Git. It includes analysis of its origin, license, ethical aspects, Linux footprint, ecosystem, and comparison with proprietary software. It also includes 5 shell scripts demonstrating Linux and automation concepts.

## Scripts

### Script 1 – System Identity Report
Displays system information such as kernel version, user, uptime, and distribution.

### Script 2 – FOSS Package Inspector
Checks if Git is installed and displays its details.

### Script 3 – Disk and Permission Auditor
Analyzes system directories for size, permissions, and ownership.

### Script 4 – Log File Analyzer
Counts occurrences of a keyword in a log file and shows matching lines.

### Script 5 – Open Source Manifesto Generator
Generates a personalized manifesto based on user input.

## How to Run

1. Give execution permission:
   chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

2. Run scripts:
   ./script1.sh
   ./script2.sh

   ./script4.sh test.log error
